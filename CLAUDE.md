# Claude Code Development Workspace

This repo is Jon's personal workspace for developing and testing Claude Code configurations, extensions, and dev tooling.

## What Lives Here

- **global/** - Global Claude Code settings (`CLAUDE.md` symlinked to `~/.claude/CLAUDE.md`)
  - `CLAUDE.md` - Foundational rules for all projects
  - `statusline.ts` - Custom statusline showing context usage, model info, and git status
- **agents/** - Custom agent definitions for the Task tool
- **.claude/** - Repo-specific Claude Code settings and hooks
- **.beans/** - Issue tracking for this workspace

## Development Focus

This workspace covers:

- Skills, Plugins, Commands, Hooks, Agents
- IDE integrations and configurations
- Dev tooling experiments
- Claude Code workflow optimizations

## Behavior

When you notice sub-optimal behavior or configuration issues, proactively suggest creating a bean to track it.
